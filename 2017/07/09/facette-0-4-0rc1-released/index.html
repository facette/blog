Facette 0.4.0rc1 released

July 09, 2017 1 min

A few bugs less and some improvements later, here is the first release candidate for the 0.4.x branch of Facette:

  • Prevent panic when link resolution fails
  • Fix providers list search filter
  • Documentation update
  • UI fixes and improvements

As always, Linux packages and various pre-compiled versions are available on Github, and the image is up-to-date on the Docker Hub.

We’re heading to a stable 0.4.0 release soon. Stay tuned!

Development Team Paris, France https://facette.io/
