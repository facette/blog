Facette 0.5.0 released

July 24, 2019 1 min

After long 6 months of testing phase, we’re releasing the 0.5.0 branch of Facette.

Here are the highlights of this version (you can read the changes details and improvements in the post announcing the rc1 release):

  • Warm startup
  • Syslog logging
  • Keyboard shortcuts
  • Duration Y axis unit
  • Y axis extremes support
  • Aliases in the URLs
  • Rearranged configuration
  • UI improvements

Linux packages and various pre-compiled versions are available on Github, and the image is up-to-date on the Docker Hub.

Development Team Paris, France https://facette.io/
